# Windows quiz
## windows quiz dapat berisi hal hal berikut:
- pertanyaan acak yang dibuat oleh pengembang dan komunitas.
- beberapa level yang dibuat oleh pengembang dan komunitas.
- efek spesial (mungkin dapat berisi flash) *segera hadir*
  
## untuk berkontribusi dalam pengembangan kode sumber windows quiz kamu hanya perlu menginstall visual studio dengan workloads .net desktop development atau visual studio code atau IDE dengan dukungan bahasa pemrograman c# dan dukungan xaml



## untuk menjalankan aplikasi diperlukan .net versi terbaru silahkan download terlebih dahulu [disini](https://dotnet.microsoft.com/en-us/download/dotnet)
